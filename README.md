# Adaptive Math Tutor

A session-based adaptive tutoring system built as part of my Master's capstone project.

## Overview

This project compares a baseline tutor with fixed difficulty progression against an adaptive tutor that adjusts difficulty based on user performance within a single session.

## Tech Stack

- Unity (C#)
- FastAPI (Python)
- SQLite
